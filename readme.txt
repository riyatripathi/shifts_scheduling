# Shift Scheduling Web App

## Description
This web application allows users to manage and schedule shifts for employees.

## Prerequisites
- Node.js
- SQLite

## Installation
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create an SQLite database (shifts.db) to store shifts.
4. Run the server with [Nodemon](https://nodemon.io/): `nodemon server.js`.

## Usage
- Access the app in your browser.
- Use the "Add Shift" button to create new shifts.
- View and manage shifts on the main page.
- Use the "Remove" button to delete shifts.

## Folder Structure
- `/views`: EJS templates.
- `/routes`: Express routes.
- `server/server.js`: Main application file.

## Technologies Used
- Node.js
- Express.js
- SQLite
- Bootstrap
